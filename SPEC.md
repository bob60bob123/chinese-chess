# 中国象棋 HTML 应用规格文档

## 1. 项目概述

- **项目名称**: 楚汉争雄（Chinese Chess）
- **项目类型**: 单页 HTML5 棋类游戏
- **核心功能**: 完整的中国象棋对弈，包含所有棋子、走法规则、吃子判断、胜负判定
- **目标用户**: 休闲玩家，需要在线对弈的中国象棋爱好者

---

## 2. 棋盘与棋子规格

### 2.1 棋盘

- 10 行 × 9 列的交叉点网格
- 两侧各有**九宫格**（将/帅、士、象/相的活动范围）
- 中间横线第 5 行和第 6 行之间为**楚河汉界**，中间有"楚河""汉界"字样
- 交叉点用细线连接

### 2.2 棋子

每方 7 种共 16 枚棋子：

| 棋子 | 数量 | 红方 | 黑方 |
|------|------|------|------|
| 帅/将 | 1 | 帅 | 将 |
| 士/仕 | 2 | 士 | 仕 |
| 象/相 | 2 | 相 | 象 |
| 车/俥 | 2 | 车 | 俥 |
| 马 | 2 | 马 | 馬 |
| 炮/砲 | 2 | 炮 | 砲 |
| 兵/卒 | 5 | 兵 | 卒 |

### 2.3 棋子外观

- 显示对应汉字（帅/将/士/仕/相/象/车/俥/馬/马/炮/砲/兵/卒）
- 红方：红色字体，黑色背景
- 黑方：黑色字体，红色背景
- 圆形棋子，带边框

---

## 3. 走法规则（完整）

### 3.1 帅/将（General）

- 只能在九宫内移动
- 每步只能走直线一格
- 不能出九宫
- 不可将帅/将同一直线上照面（面对面对视）

### 3.2 士/仕（Advisor）

- 只能在九宫内斜线移动（每步一格，斜走）
- 每方两个士，共四个斜方向

### 3.3 象/相（Elephant）

- 走直线两格（田字），且不能跳过中间的棋子
- **塞象眼**: 象走"田"字，对角中心点有棋子则不能走
- 不能过河，只能在本方半边

### 3.4 车/俥（Chariot）

- 走直线，可以横走或竖走
- 步数不限，但不能跳过途中棋子
- 移动和吃子方式相同

### 3.5 马（Horse）

- 走"日"字（一格直线 + 一格斜角）
- **蹩马腿**: 直线第一步的中心点有棋子则不能走该方向
- 八个方向均可

### 3.6 炮/砲（Cannon）

- 走直线，步数不限
- 吃子时：中间必须隔**一枚棋子**（炮架），然后攻击目标
- 不吃子时：路径上不能有棋子阻挡

### 3.7 兵/卒（Soldier）

- **过河前**（未过楚河汉界）：每步只能向前走一格
- **过河后**（已过楚河汉界）：可以向前或向左右横走一格
- 红方向上（从第10行向第1行），黑方向下

---

## 4. 游戏规则

### 4.1 胜负判定

- **将死**: 轮到走棋一方的将/帅无法逃脱（所有合法位置都被对方攻击）
- **困毙**: 轮到走棋时无任何合法走法
- 胜负即时判定并显示

### 4.2 应将规则

- 走棋后导致己方将/帅被攻击，必须立即解除（应将）
- 不允许"将军"时仍然走子

### 4.3 重复局面

- 允许不变作和（长打、长拦）

---

## 5. 用户界面

### 5.1 布局

```
+----------------------------------+
|          楚河  汉界              |
|  [9列 × 10行棋盘 + 两侧棋子]      |
+----------------------------------+
|  红方信息  |  操作区  |  黑方信息  |
+----------------------------------+
```

### 5.2 交互

- 点击选中己方棋子（高亮显示）
- 再次点击目标位置移动
- 点击空白/对方棋子判断是否可走
- 不可走的位置点击无反应

### 5.3 状态显示

- 当前回合（红方/黑方）
- 被将军提示
- 游戏结束弹窗（胜负结果）

### 5.4 按钮

- **重新开始**: 重置棋盘到初始状态
- **悔棋**: 撤销最后一步（可选）

### 5.5 音效

- 走子音效
- 吃子音效
- 将军提示音
- 胜利/失败音效

## 7. 技术实现

- 纯 HTML + CSS + JavaScript，无外部依赖
- 使用 CSS Grid/Flexbox 布局棋盘
- Canvas 或 DOM 绘制棋盘和棋子
- 音效使用 Web Audio API 或 Audio 元素
- 所有逻辑在前端运行，无需后端
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）

---

## 8. 验收标准

1. 棋盘正确显示 10×9 格子，九宫格和楚河汉界正确
2. 所有 32 枚棋子正确摆放
3. 每种棋子的走法规则正确实现
4. 蹩马腿、塞象眼规则正确
5. 照面吃将规则正确（将帅不能对脸）
6. 吃子判定正确
7. 将军判定正确
8. 将死/困毙判定正确
9. 回合切换正确（红先黑后）
10. 棋子选中高亮效果
11. 重新开始功能正常
12. 音效正常播放
